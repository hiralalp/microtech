# Monarch Forge & Fittings Website

## Overview
This is a professional website for Monarch Forge & Fittings, a leading manufacturer and exporter of high-quality Flanges and Pipe Fittings.

## Website Structure

```
microtech/
├── index.html              # Home page
├── about.html              # About Us page
├── infra.html              # Infrastructure page
├── products.html           # Products overview page
├── quality.html            # Quality Assurance page
├── contact.html            # Contact Us page
├── css/
│   └── style.css          # Main stylesheet (Theme color: #01ACF1)
├── js/
│   └── script.js          # JavaScript functionality
└── products/              # Product detail pages folder
    └── flanges-stainless-steel.html  # Sample product page
```

## Customization

### Changing Theme Color
To change the primary theme color, update the CSS variable in `css/style.css`:

```css
:root {
    --primary-color: #01ACF1;  /* Change this value */
    --secondary-color: #0187c1; /* Adjust this for gradient */
}
```
